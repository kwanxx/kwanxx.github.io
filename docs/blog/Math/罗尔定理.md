# 罗尔定理

设：

1）**f(x) 在 [a,b]区间内连续**

2）**f(x)可导**

3）**f(a) = f(b)**

则：

存在$\xi \in (a,b)$，使得
$$
\begin{aligned}
f'(\xi)=0
\end{aligned}
$$
**罗尔定理**：$f'(\xi)=0$，可以联想到$\xi$与极值有关，**为了证明极值点存在**



关于极值点的两个必要条件：

1） x=a为极值点 $\Rightarrow$ f'(a) = 0  或 f'(a)不存在

2）f(x)可导 且 x=a为极值点 $\Rightarrow$ f'(a) = 0

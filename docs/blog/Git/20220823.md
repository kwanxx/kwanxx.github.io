本地要建立Git的目录内，打开Shell或CMD，输入Git 初始化的相关命令：

  ```
  git init
  ```

### id_rsa 密钥权限

从一台电脑拷贝rsa密钥到另一台电脑，使用git提示：

Load key "/home/john/.ssh/id_rsa": bad permissions

那就将id_rsa修改权限值：

```shell
chmod 600 id_rsa
chmod 644 know_hosts
```
